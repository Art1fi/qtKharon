
# 🛡️ Kharon Password Manager

**Kharon** — это десктопный менеджер паролей с открытым исходным кодом, разработанный на C++ с использованием фреймворка Qt. Проект ориентирован на безопасность и локальное хранение данных с использованием криптографических стандартов.

## 🚀 Основные возможности

*   **Надежное шифрование:** Использование SQLCipher (AES-256) для защиты базы данных на уровне страниц.
*   **Генератор паролей:** Встроенный инструмент для создания криптографически стойких паролей с настройкой длины и набора символов (использование `std::mt19937`).
*   **Безопасный буфер обмена:** Автоматическая очистка скопированного пароля через 15 секунд для предотвращения утечек.
*   **Менеджмент баз данных:** Возможность создания нескольких изолированных зашифрованных контейнеров и быстрое переключение между ними.
*   **Мгновенный поиск:** Фильтрация записей в реальном времени по названию или логину без учета регистра.

## 🛠 Технологический стек

*   **Язык:** C++17
*   **Фреймворк:** Qt 6.x (Qt Widgets)
*   **База данных:** SQLite 3
*   **Шифрование:** SQLCipher (AES-256)
*   **Сборка:** CMake / Ninja

## 📦 Установка и запуск

### Предбодготовка
Для работы шифрования необходимо наличие плагина `SQLITECIPHER` в папке драйверов Qt.

### Сборка
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Art1fi/qtKharon.git
   ```
2. Откройте проект в **Qt Creator**.
3. Убедитесь, что выбран комплект (Kit), поддерживающий Qt 6.
4. Соберите и запустите проект.

## 🖥 Использование

1. **Запуск:** При старте выберите существующую базу данных или создайте новую.
2. **Авторизация:** Введите мастер-пароль. Помните, что без него восстановить данные невозможно (Zero-Knowledge архитектура).
3. **Управление:** Используйте кнопку «Добавить», чтобы сохранить новый аккаунт, и «Копировать», чтобы временно поместить пароль в буфер обмена.
